Checklist Upload Help

Use this form to upload a new *.CKL STIG checklist file or SCAP Scan XCCDF *.XML to the system for scoring, indexing, and managing your DoD STIG information. Select your system (if any) and then select up to five (10) *.CKL or *.XML files to upload at a time. Note that currently, the Nessus SCAP scan files are in beta. DoD SCAP files should work fine.

All fields are mandatory. Once Checklist per CKL file.

* Please fill in the Host field on your Checklist file CKL! All checklists are named by SYSTEM-HOSTNAME-STIGTYPE-REVISION.

i.e. PAL Platform - ca2335.myorg.navy.mil - Google Chrome Current Windows STIG - Release: 15 Benchmark Date: 25 Jan 2019

Template Upload Help

Use this form to upload a new *.CKL STIG checklist template for download and use for any system. A template is used as a starting point or a baseline for that checklist that your administrators, developers, cybersecurity personnel and managers can use to include known statuses and information for that particular checklist.

Example: you have an infrastructure package and a platform-as-a-service package that your application(s) run on. That infrastructure and PaaS have known good checklists as a baseline that application owners use as a starting point and adjust the remaining vulnerability items accordingly.

All fields are mandatory.
